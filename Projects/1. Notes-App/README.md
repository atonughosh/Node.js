# Notes Application
>Simple CLI application for managing notes.
>
>### Operations possible :
> - add a note
> - delete a note
> - view a note
> - list all notes
>
>### Commands :
> - add
> ```sh
> node app.js add -t "Title of note" -b "Body of note"
>```
> or without alias
>```sh
> node app.js add --title "Title of note" --body "Body of note"
>```
>
> - list
> - read
> - delete
> - --help
